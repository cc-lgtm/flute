# flute
1. 一个DOM和动画的个人快捷开发的库。
2. 支持DOM的添加样式和删除样式方法。
3. 可以删除DOM和追加DOM。
4. 支持事件队列。
5. 支持动画效果，旋转，移动，放大缩小，延迟和淡入淡出等功能。
6. 支持链式语法。

# 目录结构
dist      ---打包目录
node_modules      ---模块文件   
src       ---项目主目录
  -js     ---js文件
    -core     ---核心文件
      -anime.js     ---关于动画的js文件
      -dom.js     ---关于dom的js文件
      -init.js      ---初始化文件
    -index.js     ---项目入口文件
  -index.html     ---项目测试的html
package.json      ---模块的版本配置
README.md       ---用户查看文件
webpack.config.js       ---webpack配置文件

# 启动项目
1. npm install

2. npm run dev

3. webpack
